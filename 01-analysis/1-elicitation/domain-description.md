# Описание предметной области (Гончарная мастерская)

Настоящий документ содержит описание ключевых понятий, терминов и бизнес-сущностей гончарной мастерской «Глина», используемых при проектировании мобильного приложения клиента.

## Глоссарий терминов

* **Программа (Мастер-класс)** — тематический вид занятия, определяющий сложность, формат работы и базовую длительность (например, «Простая лепка для новичков», «Работа на гончарном круге»).
* **Мастер** — штатный сотрудник или приглашенный специалист мастерской, проводящий занятие и контролирующий технику безопасности.
* **Слот расписания (Сессия)** — фиксированный интервал времени (обычно 2–2.5 часа), к которому привязана конкретная Программа, Мастер, вместимость (лимит мест) и статус доступности для записи.
* **Бронирование (Запись)** — зарегистрированный в системе факт намерения Клиента посетить конкретный слот расписания. Может находиться в различных статусах (Активно, Отменено клиентом, Отменено мастерской).
* **Прокатный комплект** — набор инструментов и/или защитной экипировки (фартук), предоставляемый клиенту на время проведения занятия за отдельную плату или включенный в опции заказа.

## Бизнес-сущности и их атрибутивный состав

### 1. Программа (Program)
Базовая конфигурация предоставляемой услуги.
* `id` (UUID) — уникальный идентификатор.
* `title` (String) — название программы (например, "Новичковый круг").
* `description` (Text) — подробное описание процесса, чему научится клиент.
* `difficulty_level` (Enum) — уровень сложности (Новичок / Продвинутый).
* `base_price` (Decimal) — базовая стоимость участия.

### 2. Мастер (Master)
Карточка сотрудника, ведущего занятия.
* `id` (UUID) — уникальный идентификатор.
* `first_name` (String) — имя.
* `last_name` (String) — фамилия.
* `rating` (Float) — средняя оценка мастера на основе отзывов клиентов.
* `avatar_url` (String) — ссылка на фото профиля мастера.

### 3. Слот расписания (Slot)
Экземпляр занятия в календаре, доступный для бронирования.
* `id` (UUID) — уникальный идентификатор.
* `program_id` (UUID) — ссылка на сущность Программы.
* `master_id` (UUID) — ссылка на назначенного Мастера.
* `start_time` (DateTime) — дата и время начала занятия.
* `end_time` (DateTime) — дата и время окончания занятия.
* `total_capacity` (Integer) — общая вместимость (6 для новичков, до 10 для кругов).
* `available_seats` (Integer) — количество оставшихся свободных мест.
* `status` (Enum) — статус слота (Доступен / Заполнен / Отменен).

### 4. Бронирование (Booking)
Документ, подтверждающий запись клиента.
* `id` (UUID) — уникальный идентификатор.
* `client_id` (UUID) — ссылка на авторизованного Клиента.
* `slot_id` (UUID) — ссылка на забронированный Слот.
* `needs_rent` (Boolean) — флаг необходимости проката инструментов и фартука.
* `created_at` (DateTime) — время создания записи.
* `status` (Enum) — статус бронирования (Создано / Подтверждено / Отменено Клиентом / Отменено Мастерской).
* `cancellation_reason` (String) — текстовое поле причины отмены (заполняется в случае форс-мажора со стороны мастерской).
