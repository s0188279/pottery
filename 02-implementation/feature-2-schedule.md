# Фича: Просмотр сетки занятий на 7 дней

### 1. Цель
Отобразить актуальное расписание мастер-классов гончарной мастерской на ближайшую неделю, чтобы клиент мог выбрать удобный слот.

### 2. Требования (На основе US-003 и FR-003)
- Отображение слотов на ближайшие 7 дней от текущей даты.
- Карточка слота должна содержать: Название программы, Имя мастера, Дату/Время, Стоимость, Свободные места.
- Обработка Empty State, если слотов нет.

### 3. Отправленные промпты ИИ
**Промпт 1 (Генерация логики и UI):**
> "Создай компонент списка слотов расписания (ScheduleScreen) на основе модели данных из data-model (1).md. Захардкодь массив из 3-4 слотов на ближайшие дни (например, 'Простая лепка', мастер 'Иван', 4 свободных места). Отрендери их в виде карточек. Добавь обработку случая, если массив пустой — выведи заглушку 'Пока нет доступных занятий'."

### 4. Реализация
Реализован экран расписания. Данные берутся из мока, имитирующего ответ API `GET /api/v1/slots`, учитывая структуру сущностей Program, Master и Slot.
